h1. PlusAnchor jQuery Plugin

A jQuery plugin that scrolls to anchor links instead of jumping to them.

h2. Features
* Works with with ID jumping and anchor jumping

h2. Usage instructions

Place this in your &lt;head&gt; section.

<pre><script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.5.1/jquery.min.js"></script>
<script type="text/javascript" src="jquery.plusanchor.1.0.js"></script>
<script type="text/javascript">
$(document).ready(function(){
	$(document).plusAnchor();
});
</script></pre>
